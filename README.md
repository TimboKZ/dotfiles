# dotfiles

A collection of dotfiles for my Linux setup. Dotfile management is done using yadm.

# Prerequisites

The following packages must be installed:

* Vim
* Node.js
* NPM
* i3-gaps (with `i3status`)
* Compton
* `xfce4-screenshooter`
* Rofi

Required fonts:

* Dina
* FontAwesome (`fonts-font-awesome`)
* Siji
